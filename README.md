# 🎵 网易云音乐歌单 Prompt 分析器

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![GreasyFork](https://img.shields.io/badge/GreasyFork-点击安装-red.svg)](https://greasyfork.org/zh-CN/scripts/582998-%E7%BD%91%E6%98%93%E4%BA%91%E9%9F%B3%E4%B9%90%E6%AD%8C%E5%8D%95-prompt-%E5%88%86%E6%9E%90%E5%99%A8)

这是一个为“AI 赛博算命 / 音乐品味分析”量身打造的浏览器油猴脚本（UserScript）。它可以帮你**一键提取网易云音乐网页版歌单的详细数据**，并自动整理成极具逻辑的 Prompt（提示词），方便你直接喂给 ChatGPT、Claude、Kimi 等大语言模型，让 AI 深度解析你的内心世界。


## ✨ 核心特性

- **🚀 突破分页，全量读取**
  跳过网页端繁琐的分页限制，直接调用底层 API，上千首歌曲的超大歌单也能一次性完整读取（包含隐藏的添加时间等信息）。
- **🖱️ 极为强大的选择与过滤系统**
  - **鼠标拖拽框选：** 像操作桌面文件一样，在表格中按住鼠标拖拽即可跨滚动条快速连选。
  - **键盘组合键：** 完美支持 `Shift` 连选和 `Ctrl / Cmd` 穿插多选。
  - **灵活指令：** 支持输入特定序号范围（如 `1-50, 80, 120-160`），或在歌单/搜索结果中进行**随机抽样**。
- **💬 灵魂热评抓取（可选）**
  支持多线程并发抓取选中歌曲的**网易云热评前 3 名**，为 AI 提供绝佳的语料，对你的心理状态进行更丰满、毒舌或治愈的“锐评”。
- **⚙️ 高度自定义体验**
  Prompt 的开头导语支持完全自定义（内置 `{playlist}` 和 `{count}` 变量替换）。生成的结果可一键复制，或直接导出为 `.txt` 文件。

## 🛠️ 使用教程

1. **进入页面：** 在浏览器中打开并登录网易云音乐网页版（`music.163.com`），进入任意歌单页面（例如你的“我喜欢的音乐”）。
2. **打开面板：** 点击页面右下角新增的红白悬浮按钮 **【歌单 Prompt】**。
3. **读取歌单：** 在弹出的独立面板中，点击 `[读取当前歌单]` 按钮加载数据。
4. **选择歌曲：** 
   - 你可以通过搜索栏过滤某位歌手的歌；
   - 可以输入 `1-100` 选中前一百首；
   - 或者直接使用 **随机抽样** 抽 50 首代表作。
5. **生成与导出：** 按需调整 Prompt 头部语，选择是否抓取评论，点击 `[生成 Prompt]`，随后复制内容去“调戏”你的 AI 吧！

## 🤖 AI 玩法建议

你可以通过修改脚本右侧的 **Prompt 开头** 来实现不同的 AI 玩法（直接复制给 AI）：

- **【毒舌锐评】**：“以下是我在网易云音乐歌单中选择的歌曲。请用极度刻薄、幽默、一针见血的脱口秀风格，狠狠地嘲笑我的听歌品味，并推测我是个什么样的人。”
- **【心理学画像】**：“作为一名专业的荣格心理学派咨询师，请根据我收藏的这些歌曲、发行年份及网易云热评，深度分析我的潜在性格、最近的情感状态和应对压力的模式。”
- **【歌曲推荐】**：“分析这些歌曲的音乐流派、BPM、情感倾向，并向我推荐 10 首完全不在这个列表中、但绝对符合我品味的小众歌曲。”

## ⚠️ 注意事项

1. **账号权限：** 脚本基于你的浏览器当前登录状态请求数据。如果要读取私密歌单或“我喜欢的音乐”，请确保网页端已登录对应的账号。
2. **上下文长度限制：** 如果你一次性框选了成百上千首歌，并开启了“包含评论”功能，生成的字符数可能高达数万字。如果你的 AI 模型（如基础版 GPT-4）提示上下文超限，建议使用“随机抽样”功能选取 100 首最具代表性的歌曲，或者使用支持超长上下文的模型（如 Kimi、Claude 3.5 Sonnet、Gemini 1.5 Pro 等）。
3. **隐私与安全：** 脚本的所有网络请求仅限网易云音乐官方 API，所有数据仅在你本地浏览器处理生成，**绝对不会**上传任何第三方服务器。
